如何用 Hugo 搭建个人博客

September 25, 2019

1. 下载 hugo

点击链接下载

下载完成后解压, 解压的路径依你的喜好选择.

2. 配置环境变量

在Path里添加 hugo 的路径(解压的地方):

在命令行输入hugo version,如果显示了版本号等信息就说明安装成功了

3. 接下来按照官方文档的教程做即可

3.1 创建站点

hugo new site [你的站点名字]

3.2 安装一个不丑的主题

cd [你的站点名字]

git clone [主题链接] themes/[主题名]
// 如： git clone https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

echo 'theme = "ananke"' >> config.toml

3.3 写下第一个博客

hugo new posts/其实这不是我的第一个博客.md

随后在编辑器里按照 markdown 语法写博客就行

3.4 预览一下, 写得怎样

hugo server -D

3.5 生成静态文件

运行 hugo 命令

4. 部署

我选择 GitHub Page 来托管我的博客网站, So let’s getting started with it.

  1. 在 GitHub 上创建一个新仓库, 起名随意.
  2. 在本地进入public目录, 运行git init(这个目录是能直接被浏览器解析的静态资源(html, css 等), 我们直接上传这个目录到 GitHub 上))
  3. git add ., git commit -v, git push素质三连上传即可
  4. 再来到 GitHub 上, 在设置中找到 GitPage, 将 source 设置为 master
  5. 成功后应该如图上所示,然后静静等待一段时间后你点击生成的域名就能看到你的博客了
