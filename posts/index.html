如何用 Hugo 搭建个人博客

1. 下载 hugo 点击链接下载 下载完成后解压, 解压的路径依你的喜好选择. 2. 配置环境变量 在Path里添加 hugo 的路径(解压的地方): 在命令行输入hugo version,如果显示了版本号等信息就说明安装成功了 3. 接下来按照官方文档的教程做即可 3.1 创建站点 hugo new site [你的站点名字] 3.2 安装一个不丑的主题 cd [你的站点名字] git clone [主题链接] themes/[主题名] // 如： git clone https://github.com/budparr/gohugo-theme-ananke.git themes/ananke echo 'theme = "ananke"' >> config.toml 3.3 写下第一个博客 hugo new posts/其实这不是我的第一个博客.md 随后在编辑器里按照 markdown 语法写博客就行 3.4 预览一下, 写得怎样 hugo server -D 3.5 生成静态文件 运行 hugo 命令 4. 部署 我选择 GitHub Page 来托管我的博客网站, So let’s getting started with it.
